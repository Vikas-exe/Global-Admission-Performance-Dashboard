# 🌍 Global Admissions Performance Dashboard

## 📌 Project Overview

This project presents a **Global Admissions Performance Dashboard** built using **Google Looker Studio** and powered by data from **Google Sheets**. The main objective of this project was to clean the original dataset, analyze key admission metrics, generate meaningful insights, and design an interactive dashboard to support data-driven decision-making.

The dashboard provides a complete view of the global applicant pipeline, helping admissions teams and leadership understand performance across countries, programs, and outreach channels.

---

## 🎯 Purpose of the Project

The goal of this project was to:

* Track total applications, admitted students, and pending cases
* Calculate and monitor the overall acceptance rate
* Analyze country-wise and program-wise performance
* Evaluate outreach counselor performance
* Identify trends, patterns, and areas for improvement

This dashboard enables leadership to make informed decisions without relying on manual reporting.

---

## 🛠 Tools & Technologies Used

* **Google Sheets** – Data storage
* **Google Looker Studio** – Dashboard creation and visualization
* Data Cleaning & Transformation techniques
* Calculated fields (Acceptance Rate %, Aggregations, Filters)

---

## 📊 Key Metrics & Insights

From the dataset analysis:

* **Total Applications:** 5,000
* **Total Admitted:** 3,256
* **Total Pending:** 1,744
* **Acceptance Rate:** 65.12%
* **Countries Covered:** 96
* **Regions Covered:** 351

### 🔎 Major Insights:

* India generated the highest number of applications.
* Business Analytics was the top-performing program.
* Information Systems showed lower performance compared to others.
* 2022 recorded the highest performance, while later years showed a declining trend.
* Outreach performance varied significantly among counselors.
* A large number of applications had no assigned outreach caller (NA), highlighting a data quality and process gap.

---

## 📈 Dashboard Features

The dashboard includes:

* KPI scorecards for quick performance summary
* Country-level application comparison (Bar Chart)
* Program performance comparison (Bar Chart)
* Admission outcomes split (Donut Chart)
* Outreach performance tracking (Ranked Table)
* Interactive filters (Country, Intake Year, Admission Status)

All visuals dynamically update based on selected filters, allowing deeper analysis.

---

## 🚧 Challenges Faced

During development, several challenges were addressed:

* Data skew due to high concentration from one country
* Handling missing (NA) outreach data
* Managing long program names in charts
* Ensuring all filters controlled every visual correctly
* Maintaining clean layout while presenting detailed information

---

## 💡 Recommendations

Based on the analysis:

* Improve outreach coverage to reduce unassigned applications
* Focus marketing efforts on low-performing countries
* Provide training support to lower-performing counselors
* Monitor program demand trends regularly
* Track intake-year performance to identify declining patterns

---

## ✅ Conclusion

This project demonstrates how raw admission data can be transformed into meaningful insights through proper cleaning, analysis, and visualization.

The Global Admissions Performance Dashboard provides a structured, interactive, and executive-level overview of admissions performance, enabling better strategic planning and improved enrollment outcomes.
